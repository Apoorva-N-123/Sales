# Sales (ASP.NET MVC)

A web-based solution for storing and managing sales data, built using:

- ASP.NET MVC
- C#
- Microsoft SQL Server

## Features

- Add and manage sales records
- View sales history
- Update and delete entries
- Simple and clean UI

## Technologies Used

- ASP.NET MVC (C#)
- Microsoft SQL Server
- Entity Framework
- Visual Studio 2022
